# SAE401

Clone du projet Symfony/Vue
```bash
git clone https://github.com/aurelienfvre/SAE401.git ecolearn
```
```bash
composer install
```
```bash
npm install
```
```bash
npm install axios
```
```bash
npm install vue-router@4 
```
Pour lancer le serveur du projet
```bash
synfony serve:start
```
```bash
npm run watch
```
URL du projet
```bash
http://localhost:8000
```
